**[return-lastfm-shoutbox](https://github.com/faustseele/return-lastfm-shoutbox)** — браузерное расширение, возвращает inline-шауты на страницы Last.fm.

**Glider** · *скрыт* · на **CWYS** — минимальная модульная Telegram-соцсеть на mtcute.

**CWYS**/Control What You See · *скрыт* — методология + семантический middleware: разбор супер-аппов на независимые UI-модули, пользователь сам выбирает, что видеть. Glider — первый сервис на её базе; будут и другие.

**Diane** · *скрыт* — чат-стайл PKM без трения — нулевой вход. Атомарные сообщения с тегами, offline-first синк, никакой иерархии.

**[tg-scrobbler](https://github.com/faustseele/tg-scrobbler)** — Telegram-бот, скроблит любой аудиофайл, который ему отправишь. Плюс ежедневные муз-рекомендации.

**[mmpy.chat](https://github.com/faustseele/mmpy.chat)** · [дизайн](https://www.figma.com/design/SaTdkvEMsWoRl2dZn7S9Ab/mmpy-chat?node-id=0-1&t=PrP08m0m5Cfj2EMi-1) — proof of concept самописного reactivity-фреймворка в форме real-time мессенджера.
